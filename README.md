HUMAN CARE is a smart healthcare management system built using HTML, CSS, and JavaScript. It helps users register easily, book online appointments, view doctor recommendations, and navigate hospital services through a simple map interface. The project focuses on creating a user-friendly and educational platform for both patients and students.
